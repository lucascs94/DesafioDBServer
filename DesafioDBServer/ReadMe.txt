1. Para a execução do projeto é necessário ter instalado:
- Possuir a IDE Eclipse
- Plugin do Cucumber na IDE
- Gradle build tool

2. E realizar o download do projeto no repositório:
 - https://github.com/lucascs94/DesafioDBServer.git

3. Dentro do projeto, executar a classe 'VendaParaClienteSemCadastroTest.java'
	dentro do pacote 'src\test\java' para realizar a execução do cenário de teste.

4. Em caso de 'e-mail já cadastrado' durante a execução, 
	acessar a classe 'VendaParaClienteSemCadastro.feature' 
	e alterar o endereço de e-mail no step 'E preencho o email e inicio o cadastro'
	e reiniciar a execução
